# mobility
Simulation scripts for the mobility management of UAV base stations project mainly built for paper https://dl.acm.org/citation.cfm?id=3308964. 

# Requirements  
* python2.7  
* numpy
* tensorflow  
* IPython  
* matplotlib   

# Files
* main.py   
* channel.py  
* mobile_env.py  

* main_test.py  

# Build virtual environment  
` virtualenv env  `  
` source env/bin/activate ` 

# Run training  
` mkdir train `   
` python main.py  `
